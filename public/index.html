DevOps Project

During the current semester of my course, we, as a group, developed and deployed an IoT environment centered around scanners. These scanners communicated with an MQTT broker, which interfaced with a custom Pub/Sub system we built using Python. The data was then transmitted through Apache Kafka to Prometheus for metrics storage and monitoring, and subsequently visualized using Grafana.

For the first release, we utilized KinD (Kubernetes in Docker) to create virtual machines for both test and staging environments, allowing us to simulate real-world deployment scenarios. To ensure high availability (HA), we implemented StatefulSets, providing resilience and stability to our services. Additionally, we set up a CI/CD pipeline using Argo CD, automating the deployment process from development to production, ensuring efficient and reliable delivery of new updates.

In addition to the technical implementation, we used Jira for project management, where we tracked tasks, progress, and sprints to ensure smooth development. For documentation, we leveraged Confluence to maintain detailed records of system architecture, design decisions, and deployment processes. Communication within the team was facilitated through Slack, allowing for quick updates, collaboration, and issue resolution in real time.
